在 Time.H 里面没有发现 write() 的定义，用 OpenFOAM C++ Source Code Guide 进行查找，根据继承关系，发现这里的 Time.write()函数继承于 regIOobject 类的 write() 虚函数，regIOobject 类的源程序位于 $FOAM_SRC/OpenFOAM/db/regIOobject 下，虚函数 write() 内容如下

objectRegistry 类使用虚函数来实现一个通用的框架。 事实上，当程序运行至 runTime.write() 时，下面几个步骤将被执行：

  1. runTime对象调用 regIOobject::write()
  2. regIOobject 调用虚函数 writeObject(), 通常情况下这会调用 objectRegistry::writeObject(), 但是 Time 重载了这一方法，因此实际上是 runTime 对象调用了 runTime::writeObject()
  3. 在调用通常的 objectRegistry::writeObject() 之前，runTime::writeObject() 首先在位于 [caseName]/[timeName]/uniform 路径写一个名为 time 的 IOdictionary 文件
  4. objectRegistry::writeObject() 遍历其所有的子regIOobject，并调用它们各自的的 writeObject(), 如果子对象是一个普通的 regIOobject, 则调用 regIOobject::writeObject()
  5. regIOobject::writeObject() 会打开 OFstream 并调用函数 writeData(), 该函数必须在所有的 regIOobject 中定义。
